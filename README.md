# Task Manager App

A simple web-based task manager application that helps users organize and manage their daily tasks.  
The app also shows **real-time motivational quotes from an external API** to keep users motivated.

## Live Demo
https://task-manager-psi-ebon-27.vercel.app/

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Real-time motivational quotes using an external API
- Demonstrates DOM manipulation, event handling, and API integration

## Technologies Used
- HTML
- CSS
- JavaScript
